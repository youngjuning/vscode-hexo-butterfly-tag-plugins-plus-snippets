# Hexo Butterfly Tag Plugins Plus Snippets

本插件是和 Butterfly 主题的插件 [hexo-butterfly-tag-plugins-plus](https://akilar.top/posts/615e2dec/) 配套的。

## 用法

敲入`>`，然后会有提示，自动补全。

## 功能

- `>text`: Insert Text.
- `>span`: Insert Span.
- `>p`: Insert Paragraph.
- `>note`: Insert Note.
- `>tip`: Insert Tip.
- `>link`: Insert Link.
- `>hide`: Insert Hide.
- `>folding`: Insert Folding.
